3D Generators - Tree and Shrub Generator

Modeling

This generator can create trees started from the trunk, and you can modify details of each layer of branches. And for Beginners, there is a random seed to generate trees automatically.

Most of UV pieces are overlapped, I reused a lot of the space of UV map, although this will cause more seams, this allows us to reuse textures for different 3D models. Only when we change the tree style will we need to rebake maps.

Shaders

Shaders for grasses and trees.

I select billboard shader to reduce polygons, because in this shader, the surface will be always rendered face to camera, which means less polygons required for each grasses.

Both of the tree and grass can be pushed by wind. For grasses, they move with the UV offset, while for trees, they move with the offset of vertex position.
